# キャラクターの創造
アルカディアの平和を守る「掃除屋（イレイサー）」として、自身の分身となるキャラクターを作成する手順を説明する。

## 1. 筐体（リグ）を選ぶ
フルダイブに使用する戦闘用デバイスである「筐体」を1つ選ぶ。
筐体によって、初期の能力値への修正（得意な能力値に+1、苦手な能力値に-1）が決定される。
また、選んだ筐体に応じて「固有筐体スキル」を自動的に1つ修得する。

## 2. プラグインを2つ選ぶ
自身の戦闘スタイルを決定する拡張データ「プラグイン」を2つ選んでインストールする。
同じプラグインを2つ選ぶことはできない。
インストールしたプラグインによって、ステップ3で修得できるスキルの選択肢が決まる。

## 3. スキルを4つ修得する
ステップ2でインストールしたプラグインの「プラグインスキル」の中から、合計で4つのスキルを選んで修得する。
例外: 筐体に「キマイラ」を選んでいる場合のみ、自身がインストールしたプラグイン以外のプラグインスキルを1つだけ修得することができる（この特例で修得するスキルも含めて合計4つ）。

## 4. 能力値の調整
キャラクターの基礎能力を調整する。
任意の能力値（フレーム、ニューラル、ロジック、ウィル）に、合計3点の修正を割り振る。
1つの能力値に集中させても、分散させてもよい（ただし初期作成時の能力値の上限は9とする）。

## 5. パーソナルデータの設定
キャラクターの個人的な背景を設定する。

* **キャラクター名**: 好きな名前を設定する。
* **性別**: 自由に設定して構わない。
* **年齢**: 常識的な範囲で設定する。
* **理由**: 掃除屋になった理由、あるいはアルカディアを守る理由。`（正義 / 愛 / 富 / 名誉 / 居場所 / スリル）`

## 6. サブステータスの算出
決定した能力値をもとに、以下のサブステータスを計算する。

* **最大HP**: `(フレーム × 4) + ウィル + 10`
* **イニシアティブ**: `ニューラル + ロジック`
* **リソースダイス**: 初期値は `7` 個。
